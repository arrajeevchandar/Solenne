- [ ] Phase 5 exit gate passed

| 1 | 5 insight templates in worker (T1–T5) |

## Exit Criteria

- [ ] ≥1 insight visible with evidence linking source journals
- [ ] Unread badge clears on open
- [ ] No clinical language in generated insights (blocklist pass)
- [ ] Flow: login → dashboard → insight → source journal works
- [ ] Insights hidden when baseline confidence < 0.6
